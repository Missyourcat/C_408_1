## 第一章：数据的类型、输入、输出
- 基本数据类型：`int`, `float`, `char`, `double`
- 格式化输入输出：`scanf`, `printf`
- 类型转换与取值范围

## 第二章：运算符与表达式
- 算术运算符、关系运算符、逻辑运算符
- 运算符优先级与结合性
- 表达式求值顺序

## 第三章：选择、循环
- 选择结构：`if`, `else if`
- 循环结构：`for`, `while`,
- 跳转语句：`break`, `continue`

## 第四章：一维数组和字符数组
- 一维数组定义与初始化
- 字符数组与字符串处理函数
- 注意数组越界和字符串结束符 `\0`

## 第五章：指针
- 指针变量与地址运算符 `&`
- 指针与数组的关系
- 指针的基本操作、指针函数、指针数组

## 第六章：函数
- 函数定义与调用
- 传值调用与传地址调用
- 递归函数、函数声明与头文件使用

## 第七章：结构体和 C++ 引用
- `struct` 定义与使用
- 结构体数组、结构体指针
- C++ 引用：王道数据结构使用C++部分解释
